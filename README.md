# Express Portfolio

## How to push changes

1) Test the project and make sure it's working

2) Commit your project
```sh
$ git status
$ git add .
$ git commit -m "My message"
```

3) Pull the project
```sh
git pull
```

4) If you pulled something, go back to 1), otherwise you can go to 5)

4) Push the project
```sh
git push
```
